## Installation

```html
<!-- CDN -->
<script src="https://cdn.rawgit.com/PaulaoDev/khan-framework/master/vendor/build/khan.min.js"></script>
<!-- OR STATIC FILE -->
<script src="vendor/build/khan.min.js"></script>
```

## Usage

```js
let khan = new $khan();
let Router = khan.Router();
let Controllers = khan.$controller;
```
